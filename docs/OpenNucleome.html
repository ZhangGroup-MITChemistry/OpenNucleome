add_ideal_potential(ideal_file, force_group, rc_tanh_ideal=0.54, dend_ideal=1000, cutoff_ideal=2.0)[source]¶

Add the Intra-chromosomal ideal potential using custom values for interactions between beads separated by a genomic distance \(d\).

The parameter rc_tanh_ideal is part of the probability of crosslink function

\[\begin{split}f(r_{i,j}) & = \frac{1}{2}\left(1+\text{tanh}\left[(r_c - r_{i,j})^{-5}+5(r_c - r_{i,j})\right] \right) \\ & + \frac{1}{2}\left(1-\text{tanh}\left[(r_c - r_{i,j})^{-5}+5(r_c - r_{i,j})\right] \right)\left(\frac{r_c}{r_{i,j}}\right)^4,\end{split}\]

where \(r_{i,j}\) is the spatial distance between loci (beads) i and j.

Parameters

  • (string (ideal_file) – The path to the ideal potential scaling factor txt file.

  • required) – The path to the ideal potential scaling factor txt file.

  • (int (dend_ideal) – labels the index of the current force.

  • required) – labels the index of the current force.

  • (int – Cutoff of genomic separation (Default value = 1000 for the 100KB model).

  • required) – Cutoff of genomic separation (Default value = 1000 for the 100KB model).

  • (float (cutoff_ideal) – \(r_c\) in the equation, a good estimation of the bond length during the simulation (Default value = 0.54 for the 100KB model).

  • required) – \(r_c\) in the equation, a good estimation of the bond length during the simulation (Default value = 0.54 for the 100KB model).

  • (float – Cutoff of Ideal potential (Default value = 2.0 for the 100KB model).

  • required) – Cutoff of Ideal potential (Default value = 2.0 for the 100KB model).

add_softcore(force_group, epsilon_sc=1.0, sigma_sc=0.5, cutoff_sc=0.56, Ecut_sc=4.0)[source]¶

We plug the tanh potential into the WCA potential to represent the excluded volume effect;

The softcore potential will only be applied to the chromosomes (type 1, 2, 3, 4).

When \(r_{sc} < r < r_{cutoff}\), use \(E_{LJ} = 4\epsilon_{sc}\left(\left(\frac{\sigma_{sc}}{r}\right)^{12} -\left(\frac{\sigma_{sc}}{r}\right)^{6} \right) + \epsilon_{rc}\); \(r < r_{sc}\), use \(E = \frac{1}{2}E_{cut}\left(1+\text{tanh}(\frac{2E_{LJ}}{E_{cut}}-1) \right)\), where \(r_{sc} \approx 0.485\).

Parameters

  • (int (force_group) – labels the index of the current force.

  • required) – labels the index of the current force.

  • (float (Ecut_sc) – energy units (Default value = 1.0).

  • required) – energy units (Default value = 1.0).

  • (float – distance units, which represents the diameter of a 100KB-resolution bead (Default value = 0.5).

  • required) – distance units, which represents the diameter of a 100KB-resolution bead (Default value = 0.5).

  • (float – distance units, the cutoff (Default value = 0.5*1.12).

  • required) – distance units, the cutoff (Default value = 0.5*1.12).

  • (float – energy units, the energy cost for the chain passing (Default value = 4.0).

  • required) – energy units, the energy cost for the chain passing (Default value = 4.0).

WholeNucleus¶

class OpenNucleome(temperature=1.0, gamma=0.1, timestep=0.005, mass_scale=1.0)[source]¶

The OpenChrModel class performes the whole nucleus dynamics based on the compartment annotations sequence of chromosomes.

The simulations can be performed using customed values for the type-to-type, intra, inter-chromosomal, chromosomes-nucleoli, and chromosomes-speckles parameters.

In our model, each component has specific type. Compartment A: 1, Compartment B: 2, Pericentromeres: 3, Centromeres: 4, Nucleoli: 5, dP Speckles: 6, P Speckles: 7, Lamina: 8
